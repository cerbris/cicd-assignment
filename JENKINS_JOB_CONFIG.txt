=======================================================
PIPELINE STAGES
=======================================================

The pipeline has 4 stages:

1. Checkout
   - Pulls code from GitHub

2. Setup Python Environment
   - Verifies python3 and pip3

3. Run Unit Tests
   - Executes 8 unit tests
   - All must pass

4. Run Application
   - Runs the calculator demo

Post-build actions:
   - Success: Sends ✅ to WebEx
   - Failure: Sends ❌ to WebEx
